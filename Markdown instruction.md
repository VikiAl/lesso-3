## Списки

Чтобы добавит ненумерованные списки необходимо пункты выделить звездоской (*). Нпример:
* элемент один
* элемент два
* элемент три

Чтобы добавит нумерованные спики, необходимо пункты просто пронумеровать. Например:
1. первый пункт
2. второй пункт
3. третий пункт
